⬇ Download extension
  1. Unzip the downloaded file into a folder (keep the folder — don't delete it after installing).
  2. Open Chrome → chrome://extensions → turn on Developer mode (top right).
  3. Click Load unpacked → select the unzipped folder.
  4. Open your Etsy Orders page — the FUSION overlay appears and new orders sync automatically.
Updating: when a new version ships, the extension icon shows a red NEW badge — click it to come back here, download, unzip over the same folder, then hit ⟳ Reload on chrome://extensions.
